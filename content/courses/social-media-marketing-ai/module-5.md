## Section 1: Automated Comment Responses (7 minutes)

### When to Automate vs When to Be Personal

Not all comments deserve automation. Here's how to decide:

**ALWAYS Respond Personally:**
- Customer complaints or negative feedback
- Complex questions requiring context
- VIP customers or influencers
- First-time commenters
- Comments with emotional language (positive or negative)
- Questions about pricing, refunds, or policies

**Safe to Automate:**
- Thank you comments ("Great post!", "Love this!")
- Repeated FAQ questions
- Simple emoji responses
- Generic compliments
- Common requests (shipping info, store hours, etc.)

**The 80/20 Rule:**
- 80% of comments fall into 10 predictable categories
- Automate these with AI-generated templates
- Personally handle the meaningful 20%

### Creating Response Templates with AI

**AI Prompt for Response Library:**
```
I manage social media for a [type of business]. Help me create a response template library for common comments.

Create templates for:
1. "Thank you" comments
2. Product inquiry questions
3. Shipping/delivery questions
4. Price/discount requests
5. Compliments about products
6. General positive feedback
7. Tag/share requests
8. Availability questions
9. Location/hours questions
10. Recommendations requests

For each template:
- Make it warm and on-brand
- Keep under 50 words
- Include a call-to-action when appropriate
- Sound human, not robotic
- Allow for easy personalization

My brand voice is: [friendly/professional/playful/expert - describe it]
```

**Example AI-Generated Templates:**

**For Product Inquiries:**
```
Template: "Great question! [Product name] is [key benefit]. You can find all the details and specs at [link]. DM us if you need help choosing! 😊"

Personalization: Replace product name and benefit based on specific question
```

**For Positive Feedback:**
```
Template: "This made our day! 🎉 We're so happy to hear [specific thing they mentioned]. Thank you for being part of our community!"

Personalization: Reference specific detail from their comment
```

**For Shipping Questions:**
```
Template: "We ship within [timeframe]! Track your order at [link] or DM us your order number for a quick update. 📦✨"

Personalization: Add order-specific info if provided
```

### Sentiment-Based Response Systems

AI can detect sentiment and route responses accordingly:

**AI Prompt for Sentiment Routing:**
```
Create a comment classification system for my [platform] account.

Classify incoming comments as:
1. Positive (compliments, thanks, love)
2. Neutral (questions, information requests)
3. Negative (complaints, concerns, issues)
4. Spam (promotional, irrelevant)

For each category, tell me:
- Should this be auto-responded or flagged for personal reply?
- What tone should the response have?
- What's the priority level?
- How quickly should we respond?

Then give me response frameworks for each category.
```

**Sentiment-Based Response Strategy:**

**Positive Comments (Auto-OK):**
- Response time: Within 2 hours
- Tone: Warm, grateful, enthusiastic
- Action: Thank them, maybe ask for UGC permission
- Template usage: 90% automated

**Neutral Comments (Semi-Auto):**
- Response time: Within 1 hour
- Tone: Helpful, informative
- Action: Answer question, provide link
- Template usage: 70% automated with customization

**Negative Comments (Always Manual):**
- Response time: Within 30 minutes
- Tone: Empathetic, solution-focused
- Action: Address concern, take offline if needed
- Template usage: 0% - always personalized

### Handling Common Questions

**AI Prompt for FAQ Responses:**
```
I get these questions repeatedly on [platform]:
1. [Common question 1]
2. [Common question 2]
3. [Common question 3]
[List 5-10 frequent questions]

For each question:
1. Write a concise, helpful response (under 100 words)
2. Include relevant links
3. Add a friendly closing
4. Suggest if I should create a permanent FAQ post
5. Tell me how to prevent getting this question so often

Make responses sound conversational, not copy-pasted.
```

### Escalation Protocols

Know when to escalate to human support:

**Auto-Escalate When:**
- Comment contains words: "angry," "disappointed," "refund," "cancel," "scam," "terrible"
- Multiple comments from same user in short time
- Mentions competition negatively
- Legal or PR-sensitive topics
- Media or press inquiries
- Influencer or verified account comments

**AI Prompt for Escalation System:**
```
Create an escalation protocol for my social media comments.

Define escalation triggers:
1. What keywords automatically flag for human review?
2. What comment patterns need escalation?
3. How do I prioritize multiple escalations?
4. What's the response timeline for each priority level?
5. Who should handle what type of escalation?

Also create holding responses I can use while escalating to appropriate team member.
```

## Section 2: AI-Assisted Customer Service (6 minutes)

### Chatbot Integration Basics

Modern social platforms support automated messaging. Here's how to set it up effectively:

**Platform-Specific Options:**
- **Instagram/Facebook:** Meta Business Suite - Instant Replies
- **Twitter/X:** Quick Replies and Welcome Messages
- **LinkedIn:** Messaging API with automated greetings
- **WhatsApp Business:** Quick replies and away messages

**AI Prompt for Chatbot Flow:**
```
I want to set up automated customer service for [platform].

Common questions I get via DM:
1. [Question category 1]
2. [Question category 2]
3. [Question category 3]

Create a chatbot conversation flow:
1. Welcome message (friendly, brief)
2. Menu of common topics
3. Responses for each topic
4. Escalation to human for complex issues
5. Business hours auto-response
6. Follow-up after resolution

Keep it conversational, not menu-driven. Limit to 3-4 exchanges before human takeover.
```

### Training AI on Your Brand Voice

Consistency matters. Your AI responses should sound like YOU.

**AI Prompt for Brand Voice Training:**
```
Here are examples of how I communicate with customers:
[Paste 5-10 examples of your actual responses]

Based on these examples:
1. Describe my brand voice (tone, style, personality)
2. Create brand voice guidelines (dos and don'ts)
3. Give me response templates that match this voice
4. Show me examples of what NOT to say
5. Suggest phrases I should use consistently

Then create 10 practice scenarios and show how I should respond in my brand voice.
```

**Brand Voice Elements:**
- **Formality Level:** Casual vs Professional
- **Emoji Usage:** Frequent, occasional, or never
- **Humor:** Witty, wholesome, or serious
- **Empathy Level:** High-touch vs efficient
- **Language Style:** Simple vs technical

### Handling Support Queries

**Common Support Categories:**

**1. Order Status**
- Auto-response: Direct to order tracking
- Escalate if: Order is late or issue reported

**2. Product Information**
- Auto-response: Link to product page/FAQ
- Escalate if: Specific technical question

**3. Returns/Refunds**
- Auto-response: Link to return policy
- Escalate if: Customer wants to process return

**4. Complaints**
- Never auto-respond
- Always escalate immediately

**AI Prompt for Support Response System:**
```
Create a tiered support response system for social media.

Tier 1 (Instant Auto-Response):
- What types of queries qualify?
- What responses to provide?
- What resources to link?

Tier 2 (AI-Assisted Human Response):
- What queries need light human touch?
- What AI-generated templates to offer?
- How to personalize quickly?

Tier 3 (Full Human Support):
- What triggers immediate human intervention?
- What's the escalation process?
- What information to collect?

For each tier, provide example scenarios and response templates.
```

### Creating FAQ Systems

Build a searchable FAQ that reduces repetitive questions:

**AI Prompt for FAQ Creation:**
```
Based on these common questions I receive:
[List your 20 most frequent questions]

Create a comprehensive FAQ system:
1. Group questions by category
2. Write clear, concise answers
3. Add relevant links/resources

### Handoff to Human Agents

Make transitions seamless:

**Good Handoff Practices:**
```
"I want to make sure you get the best help! Let me connect you with [Name] from our team who specializes in [issue]. They'll DM you within 15 minutes. 🙌"
```

**Bad Handoff:**
```
"This is beyond my capabilities. Someone else will contact you."
```

**AI Prompt for Handoff Messages:**
```
Create warm, professional handoff messages for these scenarios:
1. Technical issue needing specialist
2. Billing/refund request
3. Custom order inquiry
4. Complaint escalation
5. Partnership/collaboration request

Each message should:
- Acknowledge the customer's need
- Explain why they're being transferred
- Set expectations for response time
- Thank them for patience
- Maintain friendly tone

Give me 3 variations for each scenario.
```

## Section 3: Sentiment Analysis & Monitoring (5 minutes)

### Understanding Sentiment Analysis

Sentiment analysis uses AI to determine if comments/mentions are positive, negative, or neutral.

**What AI Sentiment Tools Analyze:**
- **Emotion:** Happy, angry, frustrated, excited, confused
- **Intent:** Question, complaint, praise, spam
- **Urgency:** Immediate attention needed vs casual comment
- **Topic:** What the comment is about

**AI Prompt for Sentiment Dashboard:**
```
Help me set up a simple sentiment monitoring system for my social media.

I'm active on: [platforms]
I want to track sentiment about:
1. My brand overall
2. Specific products/services
3. Customer service experience
4. Recent campaigns

Suggest:
1. Free/affordable tools for sentiment tracking
2. Key metrics to monitor
3. Red flags that need immediate attention
4. How to visualize sentiment trends
5. Weekly reporting format

Keep it simple - I don't have a data science team.
```

### Real-Time Brand Monitoring

Catch issues before they become crises:

**What to Monitor:**
- Brand mentions (even without tags)
- Product names
- Competitor mentions
- Industry keywords
- Customer service keywords
- Your CEO/leadership names

**AI Prompt for Monitoring Setup:**
```
Set up a brand monitoring system for:

Brand names: [list all variations]
Key products: [list them]
Leadership: [names to monitor]
Keywords to watch: [industry terms, common issues]

Recommend:
1. Free monitoring tools
2. How to set up alerts
3. What constitutes an alert-worthy mention
4. Response priority system
5. Daily vs real-time monitoring approaches

Focus on tools that work for a [solopreneur/small team/agency].
```

### Identifying PR Issues Early

**Warning Signs to Watch:**
- Sudden spike in negative mentions
- Same complaint from multiple people
- Complaints from verified accounts or influencers
- Mentions in news or media
- Employee complaints going public
- Hashtags criticizing your brand

**AI Prompt for Crisis Detection:**
```
Create an early warning system for potential PR issues.

Analyze these scenarios and tell me which ones need immediate action:
1. [Describe scenario 1]
2. [Describe scenario 2]
3. [Describe scenario 3]

For crisis-level situations:
- What are the red flags?
- How quickly should I respond?
- Who needs to be involved?
- What's my first response?
- How do I monitor escalation?

Give me a 1-10 severity rating system with action steps for each level.
```

### Tracking Sentiment Trends

**AI Prompt for Trend Analysis:**
```
I've tracked sentiment for the last 30 days:

Week 1: 75% positive, 20% neutral, 5% negative
Week 2: 70% positive, 22% neutral, 8% negative
Week 3: 65% positive, 25% neutral, 10% negative
Week 4: 60% positive, 25% neutral, 15% negative

Analyze this trend:
1. What's causing the decline in positive sentiment?
2. Is this a concerning pattern?
3. What actions should I take?
4. What questions should I ask my audience?
5. How can I reverse this trend?

Provide specific, actionable steps.
```

## Section 4: Crisis Management with AI (4 minutes)

### Detecting Potential Crises

**Crisis Levels:**

**Level 1 - Minor Issue:**
- A few negative comments
- Isolated incident
- No media attention
- Response: Address individually, monitor

**Level 2 - Emerging Issue:**
- Multiple complaints about same thing
- Starting to gain traction
- Influencer mentions negatively
- Response: Public statement, corrective action

**Level 3 - Full Crisis:**
- Widespread negative coverage
- Media picked up story
- Hashtag trending
- Leadership involved
- Response: Official statement, action plan, PR team

**AI Prompt for Crisis Assessment:**
```
Help me assess this situation:
[Describe what's happening]

Evaluate:
1. Crisis level (1-3)
2. Potential for escalation (low/medium/high)
3. Immediate actions needed
4. Who needs to be involved
5. Timeline for response

Then provide:
- Crisis response template
- Monitoring checklist
- Communication plan
- Post-crisis review questions
```

### Crisis Response Templates

**AI Prompt for Crisis Responses:**
```
Create crisis response templates for:

1. Product defect/safety issue
2. Poor customer service experience gone viral
3. Offensive content posted by mistake
4. Data breach or privacy concern
5. Employee misconduct
6. Pricing error
7. Shipping/fulfillment disaster

Each template should include:
- Immediate acknowledgment (within 1 hour)
- Investigation update (within 6 hours)
- Resolution announcement (within 24 hours)
- Tone: Empathetic, accountable, solution-focused
- Length: Brief enough for social media

Make these authentic, not corporate-speak.
```

**Example Crisis Response Framework:**

**Immediate Response (First Hour):**
```
"We're aware of [issue] and taking it very seriously. Our team is investigating now and we'll provide an update within [timeframe]. We appreciate your patience and apologize for any inconvenience."
```

**Update (6-12 Hours):**
```
"Update on [issue]: We've identified the problem [brief explanation]. Here's what we're doing: [action steps]. Affected customers will [compensation/solution]. We'll share final resolution by [time]."
```

**Resolution (24-48 Hours):**
```
"Issue Resolved: [Summary of what happened, what we did, how we're preventing it]. Affected customers: [solution provided]. We've learned [lesson] and implemented [changes]. Thank you for your patience."
```

### AI-Assisted Rapid Response

**AI Prompt for Fast Crisis Communication:**
```
URGENT: Crisis situation.

What happened: [Brief description]
Who's affected: [Customer segment]
Current volume: [Number of complaints/mentions]

I need NOW:
1. Draft acknowledgment post
2. Internal holding response for DMs
3. FAQ for customer service team
4. Talking points for leadership
5. Media response if contacted

Keep it empathetic, brief, and solution-focused. We're a [small business/enterprise] in [industry].
```

### Post-Crisis Analysis

After the fire is out, learn from it:

**AI Prompt for Crisis Review:**
```
We just handled a crisis situation:
[Describe what happened and how you handled it]

Help me conduct a post-mortem:
1. What did we do well?
2. What could we have done better?
3. How can we prevent this in the future?
4. What systems/processes should we implement?
5. How do we rebuild trust with affected customers?
6. What's our ongoing monitoring plan?

Create an action plan to prevent similar issues and improve our crisis response capability.
```

## Section 5: Community Building Strategies (3 minutes)

### Fostering Engagement

**AI Prompt for Engagement Strategy:**
```
I want to build a highly engaged community on [platform].

Current stats:
- Followers: [number]
- Average engagement rate: [percentage]
- Active commenters: [number]

Help me create a community engagement plan:
1. Daily engagement tactics (15 min/day)
2. Weekly community spotlights
3. Monthly community events/challenges
4. Reward/recognition programs for top engagers
5. Content that sparks conversations

Make it sustainable for a [solopreneur/small team] to execute consistently.
```

**Engagement Boosters:**
- Ask questions (and actually respond to answers)
- Feature community members
- Create insider jokes/references
- Share behind-the-scenes content
- Respond to DMs personally
- Go live regularly
- Create challenges and contests

### User-Generated Content Campaigns

**AI Prompt for UGC Campaign:**
```
Design a user-generated content campaign for my [product/service].

Goals: [Increase awareness/build trust/drive sales]

Create:
1. Campaign hashtag (memorable, brandable)
2. Participation instructions (simple, clear)
3. Incentive/reward structure
4. Content prompts/ideas for participants
5. Legal/permission requirements
6. How to feature the best submissions
7. Success metrics to track

Also provide launch post, reminder posts, and thank you post templates.
```

**UGC Campaign Example:**

**Campaign:** #MyMorningWith[YourBrand]
**Ask:** Share photo of your morning routine featuring our product
**Incentive:** Featured on our page + monthly winner gets [prize]
**Hashtag:** Clear, branded, unique
**Duration:** 30 days
**Promotion:** Stories, posts, email

### Building Brand Advocates

Turn customers into cheerleaders:

**AI Prompt for Advocacy Program:**
```
Create a brand advocacy program for loyal customers.

Our most engaged followers typically:
[Describe their behaviors - frequent comments, shares, tags, purchases]

Design a tiered advocacy program:
1. How do people join?
2. What are the tiers/levels?
3. What benefits at each level?
4. How to recognize and reward advocates?
5. How to encourage them to share?

Keep it simple to manage without dedicated software. Focus on genuine relationships, not just rewards.
```

### Community Guidelines with AI

**AI Prompt for Community Standards:**
```

My brand values: [List 3-5 core values]

Draft guidelines that:

## Key Takeaways

**Remember:**
1. **Automate wisely** - Not every comment should be automated
2. **Maintain human touch** - AI assists, doesn't replace genuine interaction
3. **Monitor sentiment** - Catch issues before they become crises
4. **Have crisis protocols** - Hope for the best, prepare for the worst
5. **Build real community** - Engagement is about relationships, not metrics

**Your Action Plan:**
1. Create your comment response template library this week
2. Set up basic sentiment monitoring
3. Draft crisis response templates (hope you never need them)
4. Plan your first UGC campaign
5. Respond personally to your top 10 engagers this week
