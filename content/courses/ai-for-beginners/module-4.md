## Email & Communication

### Professional Emails
```
Write a professional email to [recipient] about [topic]. Tone: [formal/friendly]. Length: [X] words. Include: [key points].
```

### Meeting Requests
```
Draft an email requesting a meeting with [person] to discuss [topic]. Suggest 3 time slots next week.
```

### Follow-Ups
```
Write a polite follow-up email for [situation]. It's been [timeframe] since my last message.
```

---

## Meetings & Collaboration

### Agenda Creation
```
Create a meeting agenda for [topic]. Duration: [X] minutes. Attendees: [roles]. Goals: [objectives].
```

### Meeting Notes Summary
```
Summarize these meeting notes into: Decisions made, Action items (who/what/when), Next steps.
[Paste notes]
```

### Team Communication
```
Write an announcement to my team about [change]. Tone: transparent and positive. Address concerns about [X].
```

---

## Presentations & Reports

### Presentation Outlines
```
Create a 10-slide presentation outline on [topic] for [audience]. Include: opening hook, main points, conclusion.
```

### Report Writing
```
Write an executive summary (200 words) for this report: [paste content or describe].
```

### Data Explanation
```
Explain these numbers/findings to non-technical stakeholders: [data]. Make it clear and actionable.
```

## Job Search & Career

### Resume Enhancement
```
Improve this resume bullet point to highlight impact: [paste bullet]. Make it achievement-focused with metrics.
```

### Cover Letters
```
Write a cover letter for [position] at [company]. My background: [brief summary]. Why I'm interested: [reason].
```

### Interview Prep
```
Give me 10 common interview questions for [role] and suggest strong answers based on my experience: [summary].
```

### LinkedIn Profile
```
Write a compelling LinkedIn summary for [role/industry]. Highlight: [achievements]. Tone: professional but personable.
```

## Learning & Development

### Skill Gaps
```
I'm in [role] and want to advance to [next role]. What skills should I develop? Create a 90-day learning plan.
```

### Industry Trends
```
Explain the latest trends in [industry] in simple terms. What should professionals know?
```

### Professional Development
```
Suggest 5 ways to demonstrate leadership in my current [role] without being a manager.

## Time Management at Work

### Daily Planning
```
I have these tasks today: [list]. Meetings at: [times]. Help me prioritize and create a realistic schedule.
```

### Email Management
```
I have 50 unread emails. Help me create a triage system: Urgent/Important/Delegate/Archive.
```

### Project Planning
```
Break down this project into weekly milestones: [project description]. Timeline: [duration]. Team: [size].
